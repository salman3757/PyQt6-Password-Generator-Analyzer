# Installation du Générateur & Analyseur de mots de passe PyQt6

## Prérequis
- Python 3.11 ou supérieur
- Pip installé

## Étapes
1. Cloner le dépôt :
```bash
git clone https://github.com/Rafael-Iste06/PyQt6-Password-Generator-Analyzer
cd pyqt6_password_generator_analyzer
```
2. Installer les dépendances :
```bash
pip install -r requirements.txt
```
3. Lancer l'application :
```bash
python main.py
```

## Notes
* L'application fonctionne sur Windows, macOS et Linux.
* Pour un affichage correct, utiliser une police moderne comme "Segoe UI".
