# Contribuer au projet Générateur & Analyseur de mots de passe PyQt6

Merci de votre intérêt pour contribuer !  

## Règles générales
- Respectez le style PEP8.
- Commentez tout code nouveau ou modifié.
- Testez les modifications avant de proposer un PR.

## Étapes pour contribuer
1. Forkez le dépôt.
2. Créez une branche : `git checkout -b feature/ma-fonctionnalité`
3. Faites vos modifications et commit avec messages clairs.
4. Poussez votre branche et ouvrez un Pull Request.

## Bonnes pratiques
- Ajouter des tests unitaires pour toute fonctionnalité critique.
- Documenter toute nouvelle fonction dans `ARCHITECTURE.md`.
- Vérifier que l’interface PyQt6 reste réactive.
